Bandmates

Bandmates is mobile app geared toward musician groups developed with the Ionic framework. Includes a messanger using Firebase's realtime-database, a group calendar, and the ability to store event location information from the Google Places API. Lastly, it provides an archive of shows for bands, including a user input based set-list log.

Zelda Walkthrough Finder

Zelda Fan? Come find out how to get past that level you are stuck on in style. Here is a React Native mobile application that applies web scraping to find walkthroughs for any zelda level/game using a NodeJS API. Other backend packages include Express and Cheerio.

Simply Managed

Back End Capstone: Inventory management platform created with React, React-Native, NodeJS/ Express, and Postgres. Mobile app (React-Native) provides capability of scanning products and uploading sessions where user can review and edit in table format on React Web app (React). On web client, users can also add products to product list through the Amazon Product Advertising API. Communication via Node API hosted on Heroku.
